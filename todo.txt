set a new github repository and setup github pages
git init

r: install.packages("bookdown")
modify _bookdown.yml "repo" to your github repository
modify _output.yml "bookdown::gitbook/edit" to your github edition link
modify index.Rmd "github-repo" to your github repository
if you want to add more pages, add file names in both _bookdown.yml "rmd_files" and index.Rmd "Table of Contents"

sh _build.sh
git commit -a -m "publish"
git push
